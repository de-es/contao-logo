# contao-logo

`de-es/contao-logo` ist eine Erweiterung für [Contao CMS](https://contao.org/).

Sie stellt ein Content-Element und ein Frontend-Modul bereit, welches ein einzelnes Logo bzw. Bild unverändert einbindet, optional mit einem internen Link. 

Logos im SVG-Format können optional direkt in den Quelltext eingebettet werden.
